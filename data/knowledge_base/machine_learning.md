# Machine Learning

## Core Concepts
Machine learning allows systems to learn from data without being explicitly programmed. Supervised learning trains on labeled data with input-output pairs. The model learns to map inputs to outputs. Used for classification and regression. Unsupervised learning finds patterns in unlabeled data. Used for clustering and dimensionality reduction. Reinforcement learning trains an agent to take actions in an environment to maximize cumulative reward. Training set is data used to train the model. Validation set tunes hyperparameters and monitors training. Test set evaluates final model performance on unseen data. Overfitting occurs when the model learns training data too well including noise and performs poorly on new data. Underfitting occurs when the model is too simple to capture the underlying pattern.

## Key Algorithms
Linear regression predicts a continuous output as a weighted sum of input features. The model learns weights that minimize mean squared error. Logistic regression predicts probability of a binary class using sigmoid function despite the name it is a classification algorithm. Decision tree splits data on feature values to create a tree of decisions. Easy to interpret but prone to overfitting. Random forest combines many decision trees trained on random subsets of data and features taking a majority vote. Reduces overfitting. Gradient boosting trains trees sequentially each correcting errors of the previous. XGBoost and LightGBM are popular implementations. K-nearest neighbors KNN classifies a point based on the majority class of its k nearest neighbors in feature space. Support vector machine SVM finds the hyperplane that maximizes the margin between classes. Neural networks use layers of connected nodes called neurons to learn complex patterns.

## Neural Networks
A neural network consists of an input layer hidden layers and an output layer. Each neuron computes a weighted sum of inputs plus a bias then applies an activation function. Activation functions introduce non-linearity: ReLU Rectified Linear Unit outputs max of 0 and x and is most commonly used in hidden layers. Sigmoid squashes output between 0 and 1 used for binary classification output. Softmax converts outputs to probabilities that sum to 1 used for multi-class output. Backpropagation computes gradients of the loss function with respect to weights using the chain rule. Gradient descent updates weights in the direction that decreases loss. Learning rate controls step size. Too large diverges. Too small is slow. Batch size determines how many samples are processed before updating weights.

## Model Evaluation
Accuracy is the fraction of correct predictions. Not suitable for imbalanced classes. Precision is true positives divided by predicted positives. How many predicted positives are actually positive. Recall or sensitivity is true positives divided by actual positives. How many actual positives were found. F1 score is the harmonic mean of precision and recall. Used when both matter equally. Confusion matrix shows true positives false positives true negatives and false negatives. ROC curve plots true positive rate vs false positive rate at different thresholds. AUC Area Under Curve measures overall classification ability. Cross-validation splits data into k folds trains on k-1 and tests on 1 repeating k times. Gives more reliable performance estimate than a single split.

## Bias and Variance
Bias error from wrong assumptions in the learning algorithm. High bias underfits. Variance error from sensitivity to small fluctuations in training data. High variance overfits. Bias-variance tradeoff: increasing model complexity reduces bias but increases variance. Regularization reduces overfitting by penalizing large weights. L1 Lasso adds sum of absolute weights to loss function and can set some weights to zero performing feature selection. L2 Ridge adds sum of squared weights to loss function shrinking all weights. Dropout randomly disables neurons during training preventing co-adaptation and reducing overfitting. Early stopping stops training when validation loss starts increasing preventing overfitting.
